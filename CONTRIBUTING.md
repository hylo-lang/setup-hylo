# Contributing to `setup-hylo`

## Prerequisites

This action requires Node.js to be installed (we test with Node.js 24). The repo includes devcontainer configuration for easy setup.

## Development
- Install dependencies:
 > npm install

- Test:
 > npm test

- Package:
 > npm run pack

> Note: running `npm run pack` is required before submitting PRs. Unfortunately, GitHub Actions requires to check in generated files.
